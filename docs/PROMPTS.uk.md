# Приклади запитів

[Англійська версія](PROMPTS.md)

Ці приклади підходять, якщо ваш AI-помічник може запускатися за розкладом, памʼятати попередні результати й користуватися інструментами OneAIWorkers.

## Стежити за цінами конкурентів

```text
Кожного робочого дня о 09:00 використовуй OneAIWorkers, щоб прочитати ці сторінки з цінами: ...
Порівняй їх із тим, що ти памʼятаєш із попередньої перевірки.
Якщо ціна або пропозиція змінилась, надішли мені коротке повідомлення.
Потім запамʼятай новий результат для наступного разу.
```

## Стежити за сторінкою вакансій

```text
Кожні 6 годин читай цю сторінку з вакансіями.
Якщо є нові позиції про React, AI, автоматизацію або Cloudflare Workers, надішли мені короткий підсумок і посилання.
Памʼятай, що ти вже показував.
```

## Перевіряти, чи працює сайт

```text
Кожні 30 хвилин перевіряй головну сторінку і /api/health.
Якщо щось не працює двічі підряд, надішли мені повідомлення зі кодом відповіді і часом відповіді.
```

## Щоденний короткий огляд

```text
Щоранку читай ці RSS-стрічки й сторінки.
Надсилай мені тільки нове й важливе.
Памʼятай, що ти вже надсилав.
```

## Запускати webhook щотижня

```text
Щопонеділка о 09:00 викликай цей Make webhook з повідомленням: "Prepare weekly client follow-ups".
Якщо він не спрацює, надішли мені повідомлення.
```
