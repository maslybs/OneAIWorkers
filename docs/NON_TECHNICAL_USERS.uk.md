# Для нетехнічних користувачів

[Англійська версія](NON_TECHNICAL_USERS.md)

OneAIWorkers — це невеликий Cloudflare Worker. Він дає вашому AI-помічнику безпечні дії.

Це не окремий застосунок із панеллю керування.

AI-помічник памʼятає дані, перевіряє їх пізніше й вирішує, що важливо. Worker тільки виконує дії.

## Що можна робити

Ви можете попросити AI-помічника:

- перевіряти, чи працює ваш сайт;
- стежити за сторінками конкурентів;
- читати RSS-стрічки;
- викликати webhook у Make, Zapier або n8n;
- надсилати вам повідомлення в Telegram, Discord або Slack.

## Що потрібно

Мінімум:

1. Обліковий запис Cloudflare.
2. OneAIWorkers, розгорнутий у Cloudflare.
3. `/mcp` посилання, підключене до ChatGPT або іншого MCP-клієнта.

Додатково:

- `MCP_SHARED_SECRET` для приватного доступу;
- секрети Telegram, Discord, Slack або webhook для повідомлень;
- Cloudflare API token тільки якщо потрібно створювати дочірні Workers.

## Просте налаштування

1. Натисніть **Розгорнути в Cloudflare** в README.
2. Розгорніть Worker.
3. Скопіюйте `/mcp` посилання.
4. Додайте посилання в ChatGPT або інший MCP-клієнт.
5. Додайте додаткові секрети, якщо потрібні повідомлення або приватний доступ.

## Перший запит

```text
Використай OneAIWorkers. Покажи, які інструменти в тебе є, і запропонуй три прості автоматизації, які можна запускати за розкладом.
```
